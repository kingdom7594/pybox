如何用 Python 实现一个 LRU 缓存？
可以使用 functools.lru_cache 装饰器，或者手动实现双向链表 + 哈希表：
from collections import OrderedDict class LRUCache: def __init__(self, capacity): self.cache = OrderedDict() self.cap = capacity def get(self, key): if key not in self.cache: return -1 self.cache.move_to_end(key) return self.cache[key] def put(self, key, value): if key in self.cache: self.cache.move_to_end(key) self.cache[key] = value if len(self.cache) > self.cap: self.cache.popitem(last=False)
OrderedDict 内部维护了双向链表，move_to_end 是 O(1) 操作。
帮我写一个快速排序
以下是一个原地分区的快速排序：
def quicksort(arr, lo=0, hi=None): if hi is None: hi = len(arr)-1 if lo < hi: p = partition(arr, lo, hi) quicksort(arr, lo, p-1) quicksort(arr, p+1, hi) def partition(arr, lo, hi): pivot = arr[hi]; i = lo-1 for j in range(lo, hi): if arr[j] <= pivot: i += 1; arr[i],arr[j]=arr[j],arr[i] arr[i+1],arr[hi]=arr[hi],arr[i+1] return i+1
平均 O(n log n)，最坏 O(n²)。随机选择 pivot 可缓解最坏情况。
